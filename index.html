Technologies utilisées :
  • Python 3.12
  • SQLite
  • SQLAlchemy
  • Flask
  • API Swapi
Fonctionnalités :

Favoris : Enregistrement des films préférés pour un accès rapide.
Connexion : Authentification pour les utilisateurs et les administrateurs.
Statistiques : Consultation du nombre de favoris par film pour les administrateurs.
Interface : Interface simple pour la consultation des films et des favoris.

Projet Infra Res

Ce projet de groupe, réalisé dans le cadre du cours de systèmes et réseaux à l'EPSI, vise à concevoir une architecture client-serveur en environnement virtuel avec VMWARE. L'objectif est de créer une maquette de gestion de réservations d'hôtel hébergée sur un serveur web (utilisant WAMP, MySQL et phpMyAdmin). Un contrôleur de domaine sous Windows Server 2016 sera mis en place pour renforcer la sécurité, gérant l'authentification via Active Directory ainsi que des services DNS et FTP. Une machine virtuelle cliente servira à tester l'authentification et l'accès à la plateforme de réservation.

Représentation de l'infrastructure avec Cisco Packet Tracer :

Travailler en mode projet

Utilisation d’outils collaboratifs

Le suivi des tâches et la coordination de l’équipe ont été facilités par des outils comme Trello ou un tableau de bord similaire.

Répartition des responsabilités

Chaque membre de l’équipe a eu des tâches définies, telles que l’installation de WAMP, la configuration d’Active Directory ou la mise en place de services FTP.

Phases du projet

Le projet a été divisé en phases distinctes :

  • Configuration des machines virtuelles sous VMware.
  • Installation et configuration des services réseau (DNS, FTP).
  • Développement de la plateforme web de réservation avec WAMP et MySQL.
  • Tests d’authentification via Active Directory et validation des fonctionnalités.

Gérer le patrimoine informatique

Mise en place d’une infrastructure pérenne

Création d’une architecture client-serveur robuste avec un contrôleur de domaine sous Windows Server 2016 et Active Directory pour centraliser l’authentification et gérer les droits d’accès.

Maintenance et documentation

Documentation complète des étapes de configuration des services (DNS, FTP, Active Directory et WAMP) pour assurer la pérennité de l’infrastructure.
